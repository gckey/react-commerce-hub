# React-Commerce-Hub

An online shopping app, structured as a Single Page Application (SPA) with a single HTML file and React powered routing across multiple pages: HomePage, CheckoutPage, OrderPage and TrackingPage. Local mock data (products, cart, and orders) drives the UI, with Git used throughout to track every change.

# Features

- 🛒 **Add to Cart** — Browse products and add them to your cart with a selected quantity.
- 🧾 **Checkout** — View all cart items, choose a delivery option per item, and watch the order total, shipping, and estimated tax calculate in real time before placing your order.
- 📦 **Orders** — After placing an order, view your full order history with product details, delivery dates, and a one-click **Track package** button per item.
- 📍 **Order Tracking** — See a live delivery progress bar (Preparing → Shipped → Delivered) based on the estimated delivery date, with a link back to the Orders page.


# Technologies Used

- React + Vite, Node.js
